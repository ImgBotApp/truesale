Introduction

Absolute is a responsive admin template powered by Twitter Bootstrap Framework. Absolute-admin has a simple yet powerful set of UI elements and plugins to get your work done in the fastest time possible, with a user friendly and clean design to appeal to your customers.

We really care for our buyers and so in case if you have any question or feedback, please feel free to get back to us.

Structure

We have tried to follow the standards and modular structure while developing the theme. Following sections are explaining the theme File & Folder, structure, html file struture and js.

File & Folder Structure


├── Absolute-Admin main folder
├── assets/css/
│   └── All css files.
├── assets/plugins/Bootstrap/
│   └── All Bootstrap files including css & js files
├── assets/images/
│   └── All images
├── assets/plugins/font-awesome/
│   └── Font-awesome icon fonts pack
└── assets/js/
    └── All Javascripts files

HTML Structure

CSS

This template using the lot of css files, some of are from third parties. The main css files are bootstrap, style.css, font-awesome, these files are using into all html files and must be included to work it properly.

Files are explained below:

File Description
bootstrap.css Absolute-Admin uses the bootstrap v3.3.7. The core bootstrap file is being used in all the pages.
Font-awesome.css This file is using for the icon fonts
Style.css We are using the single css file for this template, this file is main css file of this template.

Javascript

Absolute-Admin uses jQuery, Bootstrap JS framework(at its core) and some of the third-party plugins. There are may more third party plugin which you can use according to your needs. The css is already containing matching style for these plugins so you will not need to do anything around it.

Files are explained below:

File Description
jquery.js, bootstrap.js, , etc. These files are used at core of the theme.
custom.js You can use this file for add any jquery scripts
